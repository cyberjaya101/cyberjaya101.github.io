# Security Policy

This is a static personal portfolio site (Jekyll, hosted on GitHub Pages) with
no backend, no forms, no cookies, and no user data collection.

If you find a security issue (e.g. a CSP bypass, a dependency with a known CVE,
or a content/UI issue that could mislead a visitor), please email
rahin577@gmail.com rather than opening a public issue. I'll acknowledge within
a few days.
