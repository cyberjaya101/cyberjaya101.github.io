# cyberjaya101.github.io

Personal portfolio + blog for Md Rahinul Islam Bhuiyan (Rahin) — AI security,
interpretability, and LLM infrastructure. Built with Jekyll, hosted on GitHub Pages.

## Local preview

```bash
bundle install
bundle exec jekyll serve
# open http://localhost:4000
```

## Structure

- `index.html` — landing page with featured "Recent Exciting Things!" grid
- `about.md`, `projects.md`, `blog.md` — main pages
- `_posts/` — blog posts
- `_layouts/` — shared HTML templates
- `assets/css/style.css` — colah-inspired styling
- `assets/img/` — project screenshots
